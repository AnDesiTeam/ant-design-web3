## 使用方法

使用图标组件，你需要安装 [@ant-design/web3-icons](https://www.npmjs.com/package/@ant-design/web3-icons) 图标组件包：

```sh
# npm
npm install @ant-design/web3-icons --save

# yarn
yarn add @ant-design/web3-icons

# pnpm
pnpm install @ant-design/web3-icons --save
```
